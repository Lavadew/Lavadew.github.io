Paul Hype Page Website Revamp

Background Info

Paul Hype Page is a company that helps foreigners or singapore citizens to start up their company in Singapore. Their aim is to provide as many information as possible to their customers.

Role

  • To Improve user experiences

  • To give the website a new look

Background Info

I was assigned to Paul Hype Page for internship and I learn a few things along the way that re-confirm my decision of wanting to become a designer. Paul Hype Page is a corporation company, not a design firm. Therefore even though I didnt learn much as a designer, I was able to get a feel of what it is like to work as a graphic designer.

Objective

To revamp the whole website and improve user experiences for the client. To also implement a solution to lessen information from flooding the entire website

Our Research

Research Goals

  • To understand the current trends for a corporate website

  • To understand the user experiences and user-flow

  • Learn about Competitors

Insights

The problem with the old Paul Hype Page website, it was flooded with information on every page and have no visual hierarchy which makes user confuse. They are also limited/stuck with a Wordpress Theme which does not allows them to customize the look as much and It is hard to customize due to the fact that there is already so much content being put into the website. Therefore I have to find ways to improve the website within the limitation



During my time in my internship, I was able to learn more about Visual Hierarchy, the spacing between buttons and the overall layout for a clean website



Competitors:


Osome I learned that Osome have a relatively modern website and have a very clean layout for easy navigation as compared to the old Paul Hype Page Website.

Art Direction

Things to do

To modernize the website, use higher quality image as opposed to low quality stock images, less words & more images/vector, implement visual hierarchy and colour system.

Art Direction

Website Concept Ideas

Final Outcome

I was able to help improve Paul Hype Page Website and help them by suggesting them to use modern looking vectors from unDraw to make their website less plain and less serious. Instead making it look more friendly and approachable. I also utilize Sans-Serif Fonts more to make it the website more readable
